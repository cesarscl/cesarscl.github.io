# Mi Ecommerce

Proyecto de maquetación de una página de e-commerce desarrollada con HTML y CSS. La pagina muestra una tienda simple con barra superior, menú lateral de categorías, grilla de productos y footer.

## Cómo visualizar el proyecto

Puedes acceder directamente al proyecto a traves de la direccion:

https://cesarscl.github.io


## Vista general

El sitio incluye:

- Layout principal construido con CSS Grid.
- Header con el nombre de la tienda.
- Sidebar con categorías y cantidad de productos.
- Sección de productos organizada en tarjetas.
- Imágenes locales para cada producto.
- Botones de compra visuales.
- Footer con identificador de la tienda.

## Tecnologías utilizadas

- HTML5
- CSS3
- CSS Grid
- Flexbox

## Objetivo del desafío

Practicar la creación de layouts con CSS Grid, combinando áreas de página como `navbar`, `sidebar`, `products` y `footer`, junto con el uso de Flexbox para alinear contenido dentro de los componentes.
