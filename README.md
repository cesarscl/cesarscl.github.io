# Calculando total JS

Proyecto desarrollado como parte del desafío **Calculando el total**, enfocado en practicar la manipulación del DOM con JavaScript.

La aplicación muestra una tarjeta de producto con su precio base y permite aumentar o disminuir la cantidad seleccionada. Cada cambio actualiza automáticamente el total a pagar.

## Objetivo

Calcular dinámicamente el total de una compra a partir del precio de un producto y la cantidad seleccionada por el usuario.

## Funcionalidades

- Muestra el precio base del producto.
- Permite aumentar la cantidad con el botón `+`.
- Permite disminuir la cantidad con el botón `-`.
- Evita que la cantidad sea menor que cero.
- Actualiza el total a pagar en tiempo real.
- Formatea los valores numéricos con formato local `es-CL`.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

1. Descarga o clona este repositorio.
2. Abre el archivo `index.html` en tu navegador.
3. Usa los botones `+` y `-` para modificar la cantidad del producto y ver el total actualizado.
